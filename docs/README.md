# 开成不公 (KCBG) - 电商溯源系统

## 项目概述

这是一个**前后端分离**的电商溯源系统，包含两个独立项目：

- **kcbg/** - PHP后端项目 (FastAdmin + ThinkPHP)
- **cwlsuniapp/** - UniApp前端项目 (跨平台移动应用)

## 核心功能

### 产品溯源系统
- 视频化产品追踪
- 产品来源记录
- 质量保证文档
- 供应链透明化

### 电商平台
- 商品管理（分类、品牌、SKU）
- 购物车系统
- 订单管理（下单、支付、售后）
- 支付集成（支付宝、微信支付）
- 物流追踪
- 优惠券系统
- 积分系统
- 用户评价

### 用户系统
- 用户注册/登录
- 微信授权登录
- 第三方登录（QQ、微博）
- 地址管理
- 个人资料管理

## 技术栈

### 后端
- **框架**: FastAdmin + ThinkPHP 5.x
- **语言**: PHP 7.1+
- **数据库**: MySQL 5.7+ (utf8mb4)
- **插件系统**: 支持动态加载插件

### 前端
- **框架**: UniApp + Vue.js
- **UI库**: uView UI
- **支持平台**: 微信小程序、H5、Android/iOS App
